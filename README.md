# Avengers Heroes

A Progressive Web App featuring profiles, stats, and info about Marvel Avengers characters — starting with Hawkeye.

## Features

- Responsive design with full viewport hero images
- PWA support for offline usage and installability
- Detailed hero stats, timeline, media, and merch sections

## How to run

Open `index.html` in your browser or deploy the app using Netlify, GitHub Pages, or any static hosting.
